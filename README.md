# Instruction Classification with BERT (Chinese)

## Requirements

Install the required packages using pip:

```bash
pip install torch transformers scikit-learn pandas numpy
```

## How to Run

1. **Clone the repository**  
```bash
   git clone https://github.com/ziling-huang2002/Assignment1-focused-on-ASR-error-correction.git
   cd Assignment1-focused-on-ASR-error-correction
```

2. **Run**
  <br>Open [`main.ipynb`](main.ipynb) using Jupyter Notebook or VSCode and execute all cells in order.
